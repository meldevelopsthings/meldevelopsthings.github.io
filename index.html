Projects

Clairvoyance Task Manager

Clairvoyance is a task management software that I designed for my university course. It is a web application made using a simple stack of HTML, Tailwind CSS, PHP and Javascript. The app uses a drag and drop functionality to allow for the sorting of tasks between multiple different lists at a time, under a kanban board view which can allow users to monitor their progress incrementally.

The main features are defined primarily by CRUD, allowing the user to interact with the app and its database through an authentication system. This system requires users to sign up to the app before they use it, but allows a lot of options for task management for anyone who does.

iLet4You CRM System

iLet4You is a letting agency based in Sheffield who, through my university course, tasked a group of us to collaborate and create a CRM system for them using C# and Winforms. This software was required to be a CRUD application that could function on an Intranet, which would allow the agency to manage their properties, landlords and tenants and all information regarding them. This would include things like a tenants rental data, like how much rent they have paid.

In the end, iLet4You was happy with what we provided in the end. We were even chosen by our university to showcase it at an event!
